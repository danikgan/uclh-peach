Choices: N3-charts


N3-charts is a JavaScript charting library built for AngularJS.


Key Features

  • Easy to setup
  • Simple, elegant and interactive graphs
  • SVG powered
  • Built on top of D3.js [1]


    • Usage

    N3-charts could be an option to handle data visualisation if we choose AngularJS for the front end application. Its greatest advantage is perhaps the fact that it can be used in conjunction with D3. This would enable us to easily implement graphs that fit the functionalities of n3-charts while retaining the D3â€™s broad customization options in other areas.


    Advantages

  • Easy to set-up provided AngularJS is the basis of the application
  • Beautiful graphs with attention to detail
  • Built on top of D3.js thus can be used in conjunction with D3
  • Well documented along with instructions and examples
    • [2]


    Disadvantages

  • Limited to Angular


    • Alternatives

    Below are some popular alternative javascript charting libraries. Some of which will also be explored in our research.

  • Google Charts
  • D3.js
  • ChartJS
