Performance summary

The website is live and accessible to anyone interested in it, including our client. However, we have encountered a few bugs to be fixed in the next term:

  • When accessing any of the “Components” section, the menu becomes unresponsive, thus, only the logo of the website can be used to go back to the home page.
  • Styles of the content do not correspond to the styles of the website.
  • “Show code” button has not been implemented and will require further installation of ReactJS algorithms and logic.
  • Some links, for instance “Show code”, lead to the official forum of Bootstrap-React community.
  • New React object can only be uploaded / updated manually.
